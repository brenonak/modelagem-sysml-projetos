# Laboratório de Engenharia de Sistemas (SysML)

Este repositório contém os projetos desenvolvidos durante a disciplina de Laboratório de Engenharia de Sistemas. O foco principal da disciplina foi a aplicação prática da **System Modeling Language (SysML)** para a especificação, análise e projeto de sistemas complexos, utilizando a ferramenta **Visual Paradigm**.

A modelagem de sistemas (MBSE - Model-Based Systems Engineering) é essencial para garantir que os requisitos, a arquitetura e o comportamento dinâmico de hardware e software estejam perfeitamente alinhados antes da implementação.

## Projetos Desenvolvidos

Os projetos exploram diferentes facetas da engenharia, desde a arquitetura interna de processadores até o ciclo de vida de produtos embarcados de consumo.

### 1. [Modelagem SysML de um Processador MIPS Monociclo](./p1-modelagem-processador-mips)
Modelagem arquitetural e comportamental de um processador de 32 bits (arquitetura MIPS monociclo, organização Harvard). O projeto utiliza diagramas de Pacotes, Requisitos, Casos de Uso e Atividades para mapear o datapath, a unidade de controle e a execução de instruções RISC.

### 2. [Modelagem SysML de um Alto-Falante Bluetooth IP68](./p2-modelagem-alto-falante-ip68)
Modelagem de um sistema embarcado (hardware e firmware) de um alto-falante portátil à prova d'água. O projeto detalha as máquinas de estado da conectividade Bluetooth, o gerenciamento de energia (bateria de Li-ion), a lógica de auto-shutdown e os estados físicos críticos de vedação (certificação IP68).

## Ferramentas Utilizadas
* **Visual Paradigm:** Ferramenta principal para a criação dos diagramas SysML (`.vpp`).
* **System Modeling Language (SysML):** Extensão da UML voltada para a engenharia de sistemas.
